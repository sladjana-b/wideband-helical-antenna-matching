Helical Antenna matching network.

This respository contains the design files for wideband impedance matching network for helical antenna.
The matching network is designed for 0.6GHz up to 10GHz.

The workflow includes: antenna S-parametar analysis, matching network design (MWO), 3D EM validation in HFSS, PCB layout preparation in KiCad and Gerber files for fabrication.
For the matching network, there was S1P parameteres used for simulating helical antenna in MWO.

## Tools:
Anysis HFSS, KiCad, AWR Microwave Office

## Status:
Work in progress; the current focus is on final PCB layout preparation and comparison between circuit level and EM simulation analysis.
For now, the simulation results show good matching from circuit analysis MWO (which is reffers to pink) and HFSS 3D EM analysis (which reffers to green line).
